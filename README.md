# Tech Crush Weather Dashboard

A modern and responsive Weather Dashboard built with HTML, Tailwind CSS, and JavaScript.

This project is being developed by Tech Crush Group 10 as a collaborative frontend project. The application allows users to search for a city and view real-time weather information, forecasts, and weather highlights through a clean and modern user interface.

---

## Project Goal

The goal of this project is to build a professional weather application that demonstrates:

- Responsive web design
- API integration
- Team collaboration using GitHub
- JavaScript DOM manipulation
- Modern UI implementation using Tailwind CSS

---

## Tech Stack

- HTML5
- Tailwind CSS
- JavaScript (ES6+)
- Weather API
- Git & GitHub

---

## Core Features

### Current Weather
- City name
- Current temperature
- Weather condition
- Weather icon
- Date and time
- High and low temperatures

### Weather Details
- Humidity
- Wind Speed
- Pressure
- Feels Like Temperature

### Forecast
- 5-Day Forecast
- Hourly Forecast

### Weather Highlights
- UV Index
- Visibility
- Sunrise Time
- Sunset Time

### Additional Features
- Search weather by city
- Current location weather
- Dark mode support
- Recent searches
- Responsive design
- Loading states
- Error handling
